# Testris
Jeu Tetris en HTML/CSS/JS : grille verticale, 7 tétriminos colorés, contrôles clavier (←→↓ déplacer, ↑ pivoter). Lignes complètes = points + disparition. Score, niveau progressif (vitesse croissante), Game Over. Bouton Start/Restart, aperçu pièce suivante, animations fluides. Code commenté pédagogiquement.
